后面讲到的棋子都是指同色棋子

五的知识点
    ...
  • 五连：在一条线上面刚刚好连成五颗棋子。
  • 五连点：能够形成五连的点
    • PS：五连否定 禁手，否定活四...五连出现棋局结束 其它没有成五的子力都被否定。
      ...

四的知识点
    ...
  • 活四：在一条线上面连成四颗棋子，在四颗棋子相连的两端各有一个五连点。
  • 活四点：能够形成活四的点
  • 冲四：在一条线上 相连的五个格子内有四颗同色棋子，剩下一个空格是这四颗棋子相关的唯一五连点。
  • 冲四点：能够形成冲四的点
  • 其它的四：没有五连点的四
      • ...

三的知识点
    ...
  • 活三：能够成为活四的三。
      • ...
    • 连活三
    • 跳活三
  • 活三点：能够形成活三的点
  • 眠三：只能够成为冲4的三。
  • 眠三点：能够形成眠三的点
  • 其它的三：即没有活四点，也没有冲四点的三
      • ...

  • 其它的二：即没有活三点，也没有眠三点的二

禁手
    ...
  • 长连禁手：黑棋一子落下,在一条线上面相连棋子在六颗以上（包括六颗), 这手棋就是长连禁手
  • 四四禁手：黑棋一子落下,在棋盘上面增加了两个(或更多)冲四或活四,这手棋就是四四禁手
  • 三三禁手：黑棋一子落下,在棋盘上面增加了两个(或更多)与这手棋相交的活三,这手棋就是三三禁手
    • PS:五连否定禁手，只有黑棋才有禁手,白棋没有禁手。禁手之间不否认，禁手出现棋局结束，禁手否定冲四，活四等...其它没有五连的子力
      ...

复杂禁手
    ...
  • 复杂三三禁手：在判断三三禁手的过程中，出现了禁手否定 活四点(或者五连否定 活四点)的禁手
    • PS:只有三三禁手,才有复杂禁手
      ...

长连禁手的判断
    ...
  1. 填子
  2. 四个方向逐个判断长连
      3. ...
    1. 在这条线上，有长连的形状
    2. 其它方向没有五连
    3. 同时满足前面条件，才是长连禁
  3. 找够一个长连，是长连禁手
  4. 没有找到长连，不是长连禁手

四四禁手的判断
    ...
  1. 填子
  2. 四个方向逐个判断
  3. 找够两个冲四（或活四），是四四禁手
  4. 不够两个冲四（或活四），不是四四禁手
    5. PS：一条线上，可能会有两个冲四。这种情况就是一条线上面的四四禁手

三三禁手的判断
    ...
  1. 填子
  2. 四个方向逐个判断活三
  3. 找够两个活三，是三三禁手
  4. 不够两个活三，不是三三禁手

被否定的子力
    ...
  • 五连否定其它子力
  • 禁手否定其它子力
    • PS：当棋盘上面出现五连（或禁手）时，棋局已经结束，其它还没有五连的子力都被否定
